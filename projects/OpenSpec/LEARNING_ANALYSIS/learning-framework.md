# Repository Architecture Learning Framework v2

## Reference System
This session uses the comprehensive learning system outlined by the user.

### Core Principles
- Analysis priority: workflow → boundaries → orchestration → quality gates → extensibility → failure modes → details
- Always bind conclusions to repository evidence
- Distinguish: [观察] [推断] [迁移] [待验证]
- Top 20% insights only
- State machine thinking, not feature lists
